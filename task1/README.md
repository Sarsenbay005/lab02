#Компоненттер
FragmentLayout.jsx - реакт фрагмент арқылы header, main, footer шығарады
ItemList.jsx - map() қолданып тізімді key арқылы шығарады
CombinedFragment.jsx - тақырып, тізім және элементтердің жалпы санын көрсетеді
App.jsx - болса таск 1 ді іске қосатын негізгі компонент
